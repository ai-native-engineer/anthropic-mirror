Lovable MCP Server lets you create, iterate on, inspect, and deploy Lovable projects directly from Claude.

You can use Lovable to:

**Build a working prototype:**  
"Create a working prototype for [example.com] in Lovable"

**Generate variations:**  
"Create 5 different versions of [an idea] in Lovable"

**Fix and restyle:**  
"Fix all the bugs in [my project] and update it to look more like [this design system]"

**Audit and clean up:**  
"List all my Lovable projects, suggest which ones to delete, and rename the rest descriptively"
