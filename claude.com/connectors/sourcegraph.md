Connect Claude to Sourcegraph's powerful code intelligence APIs - enabling semantic code search, symbol lookup, file retrieval, and repository navigation across your entire codebase, including private and multi-repo setups.

You can use Sourcegraph to:

**Search across repos:**  
"Find every call to validateToken across our org"

**Look up a symbol:**  
"Where is PaymentProcessor defined and what references it?"

**Navigate a codebase:**  
"Show me the file tree for the auth service"
