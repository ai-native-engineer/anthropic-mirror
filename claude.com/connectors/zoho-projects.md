Control Zoho Projects data and task operations through MCP tools.

You can use Zoho Projects to:

**Create and manage tasks:**  
"Create a task called API Integration Review in Project Alpha and assign it to Rahul"

**Track project progress:**  
"List all overdue tasks in Project Alpha with their owners and priority"

**Manage project phases:**  
"Update Project Alpha status to In Progress and add milestone for Q2 delivery"

**Monitor team workload:**  
"Show me all active projects and their current task counts by assignee"
