Biomni Lab is the first Integrated Biology Environment (IBE), a unified workspace where biologists collaborate with AI agents to execute complex research with rigor and at scale. This MCP connector exposes Biomni Lab's core capabilities: starting and managing research tasks, streaming agent output in real time, uploading and managing project files, and retrieving analysis results. Every task is reproducible with full code, transparent reasoning, and clear references.

You can use Biomni Lab to:

**Start a research task:**  
"Start a new task in my Genomics project to predict the structure of TP53 using AlphaFold."

**Upload data and run analysis:**  
"Upload this FASTQ file to the project, then start a task to run differential expression analysis."

**Check results:**  
"Is my variant classification task done? Show me the result files."
