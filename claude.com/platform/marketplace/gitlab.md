* Use case

GitLab is a unified DevSecOps platform used by developers, security teams, and platform teams to plan, build, secure, and ship software. GitLab Duo Agent Platform lets teams orchestrate Claude-powered AI agents across the full lifecycle, while GitLab Orbit keeps those agents grounded in your repo data, dependencies, and current project context so their actions stay accurate. Everything runs inside your GitLab environment, against your code and your controls.

## What you can do

* **Ship faster with quality.** Get AI assistance across the full SDLC, from planning through deployment. Turn work items into merge requests in minutes, get inline code reviews at scale, and diagnose and fix failed pipelines to ship with confidence.
* **Keep agents grounded in your context.** GitLab Orbit maps your code, work items, pipelines, and deployments, so agents act on real lifecycle context instead of guesswork, which means less rework.
* **Review changes with more context.** AI support on merge requests and code review helps teams catch issues earlier, before release.
* **Watch every agent's action.** Each agent action carries an identity, a policy path, and an audit record, so platform and security teams can see, attribute, and control what agents do.
* **Work in the tools your teams already use.** Connect GitLab with Claude-powered agents so developers stay productive without leaving their existing workflow.
