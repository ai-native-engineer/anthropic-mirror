### You decide what Claude can access

You choose which folders and connectors Claude can see and it can't access anything without your explicit permission.

Learn more

[Learn more](https://support.claude.com/en/articles/13364135-use-cowork-safely)Learn more

### By default, Claude asks before acting

Before taking significant actions, Claude shows you what it plans to do and waits for your approval. You can redirect, refine, or take a different approach at any step. Always confirm before Claude handles financial, personal, or work-critical tasks.

Now, you can give it permission to take action without approval. Your choice.

### For Teams & Enterprise users

Claude Cowork conversation history is stored locally on your device. Administrators can toggle it on or off at their discretion. Avoid granting access to local files with sensitive information, like financial documents.

Claude Cowork streams tool calls, file access, and approval states to your SIEM through OpenTelemetry, enabling data mapping to the Compliance API.

### Deployment flexibility

Access through a Claude account or your existing cloud provider (Amazon Bedrock, Google Cloud's Vertex Al, or Microsoft Foundry).

## FAQ

### What is Claude Cowork?

Cowork is Anthropic's agentic AI system for knowledge work. It runs on desktop, connects to local files and applications, and completes multi-step tasks from start to finish. The user defines the goal. Cowork figures out how to get there.

### What’s the difference between Claude Cowork and Chat?

In regular chat, Claude responds to your messages but can't access your files directly. In Cowork, Claude has permission to read, edit, and create files in folders you specify—so it can actually complete tasks rather than just describe how to do them.

### What's the difference between Claude Cowork and Claude Code?

Desktop app users can start a task in [Claude Desktop](http://claude.com/download) and let it handle work in the browser without switching windows by enabling Claude in Chrome as a connector. Claude Code is a command-line tool designed for developers. Cowork brings the same agentic architecture to the desktop app, designed for non-coding knowledge work. No terminal required.

### Can I assign tasks to Claude from my phone?

Yes, now in research preview for Pro and Max subscribers. You'll need both the Claude desktop and mobile apps at their latest versions, then you need to pair them. Your desktop needs to be running and your computer needs to be awake. Claude does the work on your desktop and sends you the result. This feature is rolling out gradually and may not be available to all eligible users immediately.

### What does computer use mean?

Computer use lets Claude interact directly with your screen to open apps, navigate your browser, or run tools. Claude reaches for your connectors and integrations first, falls back to your browser when needed, and only uses your screen as a last resort. This matters because using a direct integration is faster and more precise than navigating through the screen.

When Claude does interact with your screen, it asks permission before accessing each application. You see what it's doing and can stop it at any step. Computer use is in research preview for both Cowork and Claude Code.

### What kind of files can Claude Cowork read, create, edit, and analyze?

Cowork can understand the following file types:

* **Documents and text:** Word documents (.docx, .doc), PDF files (.pdf), Plain text (.txt), Markdown (.md), HTML (.html), JSON (.json), CSV (.csv), TSV (.tsv)
* **Spreadsheets:** Excel files (.xlsx, .xls, .xlsm)
* **Presentations:** PowerPoint files (.pptx, .ppt)
* **Images:** Common formats like .png, .jpg, .jpeg, .gif, .svg, .webp
* **Data and config:** YAML (.yaml, .yml), XML (.xml), TOML (.toml)
* **Notebooks:** Jupyter notebooks (.ipynb)
* **Code files:** Pretty much any programming language: Python (.py), JavaScript (.js), TypeScript (.ts), React (.jsx, .tsx), Java, C/C++, Go, Rust, Ruby, PHP, SQL, shell scripts, etc.

You can also use Connectors to view additional files.

### Does Claude Cowork use more of my rate limit?

Yes, agentic tasks consume more capacity than regular chat. Claude coordinates multiple sub-agents and tool calls to complete complex work.

### Can admins control Claude Cowork access?

Yes. Admins can toggle Claude Cowork off for their organization anytime in [Admin Settings](http://claude.ai/admin-settings/cowork). Manage Cowork access across teams with role-based access controls.
