The Dropbox plugin connects Claude to your Dropbox account so you can find, inspect, organize, share, and clean up content without leaving Claude, and it works only with files you already have access to and respects your existing Dropbox permissions.

**Key features**

* Search files and folders by name, keyword, type, or location, and browse folder and file metadata.
* Read and summarize supported file content.
* Save generated content and outputs directly back to Dropbox.
* Inspect, create, or reuse Dropbox shared links after confirmation.
* Create and inspect file requests to collect uploads.
* Organize folders by creating, copying, and moving content, with async job-status checks.

**How to use**

Install the plugin, complete Dropbox OAuth when prompted, and review the requested permissions. Then ask in plain language:

* "Find the Q4 planning deck in Dropbox."
* "Summarize this Dropbox file and tell me when it was last modified."
* "Create a shared link for the Launch folder."
* "Create a Dropbox file request for vendor invoices."
* "Review duplicate exports in this folder and ask before deleting anything."
