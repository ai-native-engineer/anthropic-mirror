Craft production-grade WordPress sites and applications. Everything from themes and plugins to commerce and deployment.

Build WordPress themes, plugins, custom blocks, and online stores with Claude. It writes clean, well-structured code, whether you're scaffolding a block plugin, making changes to a theme, adding commerce to a site, or starting a brand new project.

The plugin includes a guided site builder that takes you from an idea to a working site. There's also a site auditor that checks accessibility, design, and performance. It fixes what it finds, then re-checks until everything passes. Skills for WooCommerce and Jetpack let you add commerce, preview, and deploy.

**How to use:** Ask Claude to build WordPress projects and the right skills activate automatically. Try prompts like "I need a custom block that displays team member cards," "Help me plan a WooCommerce store for my bakery," "I want my site to feel more modern and clean," or "Check my site for accessibility and performance issues." Works with WordPress Studio for previews and deployment.
