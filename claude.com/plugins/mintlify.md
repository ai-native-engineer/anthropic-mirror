The Mintlify plugin gives Claude accurate knowledge of the Mintlify platform and direct access to your project. The plugin gives Claude component syntax and configuration options from precise reference knowledge that covers every Mintlify component, navigation pattern, and configuration setting. Claude can read and edit your content directly through Mintlify's MCP server.

Key capabilities include creating and editing pages with correct MDX and component syntax, restructuring navigation, updating site configuration, and opening pull requests with your changes. A live documentation lookup connects Claude to Mintlify's published docs for real-time reference if the built-in knowledge is ever insufficient to complete a task.

**How to use:** Ask Claude to help with any Mintlify task and the skill activates automatically. Try prompts like "Update the quickstart guide based on this user feedback", "Set up a changelog", or "Use the mint CLI to get recent user comments and identify any pages to update." For live editing sessions, start with "check out my Mintlify project." Claude will authenticate and open a preview so you can watch changes render in real time.
