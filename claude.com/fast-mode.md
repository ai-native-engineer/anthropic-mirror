# Fast mode for Claude Opus 4.8 waitlist

Fast mode for Opus 4.8 is available as a limited research preview on the [Claude Platform (API)](https://platform.claude.com/docs/en/build-with-claude/fast-mode).

Reach out to your account manager to enable access. If you do not have an account manager, you can join the waitlist for fast mode by completing this form.

Developers using Claude Code with extra usage enabled can use fast mode today. [Learn more](https://code.claude.com/docs/en/fast-mode).

## FAQ

### What is fast mode?

Fast mode is a high-speed configuration of Opus 4.8 that delivers 2.5x faster output token speeds. It provides the same Opus-level model intelligence with significantly faster speeds.

### Where is fast mode available?

Fast mode is available in research preview on Claude Code for all developers with extra usage enabled. [Learn more](https://code.claude.com/docs/en/fast-mode).

Fast mode is also available in a limited research preview on the Claude Platform. If you wish to request access, please reach out to your account managed. If you do not have an account manager, you can join the waitlist for fast mode by completing this form.

### Where can I learn more about fast mode?

Explore the documentation to learn more about fast mode on [Claude Code](https://code.claude.com/docs/en/fast-mode) and the [Claude Platform](https://platform.claude.com/docs/en/build-with-claude/fast-mode).
